# Doc Gardening (Local)

## Checklist

- `AGENTS.md` is still short and points to the right docs.
- `docs/README.md` reflects the real active state and does not treat completed boards as current work.
- `docs/knowledge/README.md` indexes stable docs that actually exist.
- `docs/history/README.md` indexes completed loops and historical references.
- Move "notes that became true" into `docs/knowledge/`.
- Archive completed loops in place under `docs/todo/` and remove them from active pointers.
- Delete or archive stale docs; avoid accumulating dead files.
